## Installation
You need installed [Node](https://nodejs.org/en/download/) and [NPM](https://npmjs.com) or [YARN](https://yarnpkg.com/) to install Profile UI.

With NPM:
```
npm install
```
With YARN:
```
yarn install
```
## Usage
```
npm start
```
or
```
yarn start
```
